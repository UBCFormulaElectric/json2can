# Conventions

## Naming Conventions

1. Message, signal, and enum names are in `PascalCase`. 
2. Enum values are in `SCREAMING_SNAKE_CASE`. 

## Message IDs

Standard CAN allows an 11-bit message identifier, so up to 2048 possible IDs.
